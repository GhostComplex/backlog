# Project Isotope

Multi-agent orchestration platform for developers.

## What is this?

A developer-facing platform for building, deploying, and managing multi-agent systems with:

- **Agent Core** — Single agent lifecycle management with unified LLM API
- **Agent Runtime** — Containerized (Docker) and non-containerized execution
- **Agent Iteration** — Persistent prompt evolution with image-based distribution
- **Async Messaging** — IM-style asynchronous inter-agent communication
- **Orchestrator** — Multi-agent session management and coordination
- **Web Dashboard** — Conversation visualization and agent management
